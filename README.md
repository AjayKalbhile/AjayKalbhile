## 🔍 Projects

### 🖥 Security Monitoring with Splunk
Built a home SIEM lab ingesting 500+ log events/day from Windows and Linux endpoints. Detected 3 brute-force attack patterns, wrote custom alert rules, and built dashboards that surfaced signal — not just noise.

> **Key takeaway:** Real SIEM work is knowing what *normal* looks like so anomalies become obvious.

📄 [View SOC Lab Summary Report →](https://github.com/AjayKalbhile/Cybersecurity-Portfolio)

---

### 🔓 Vulnerability Assessment Lab
Stood up a controlled attack environment — Kali Linux + Metasploitable VMs. Ran full network scans with Nmap, surfaced 15+ open ports, mapped them to CVEs, and delivered a structured Hardening Report with remediation steps.

> Not just scanning. Reports a real security team could act on.

📄 [View System Hardening Report →](https://github.com/AjayKalbhile/Cybersecurity-Portfolio)

---

### 🎣 Phishing Analysis Project
Investigated 10+ real phishing samples. Extracted IOCs — malicious URLs, IPs, sender infrastructure — using VirusTotal and manual header analysis. Mapped attacker TTPs to MITRE ATT&CK framework. Also designed a full phishing simulation campaign for a Mastercard security awareness exercise.

📄 [View Threat Report →](https://github.com/AjayKalbhile/Cybersecurity-Portfolio)
